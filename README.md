# lee-ocr的安装与使用

## 安装依赖
- python >3.6
  - opencv-python 4.5.1
  - pyQt5
  - pyinstaller
  - PIL
  - numpy
- tesseract

## 使用

- shell终端
```bash
python lee_ocr.py
```
- pycharm或vscode 运行lee_ocr.py即可
